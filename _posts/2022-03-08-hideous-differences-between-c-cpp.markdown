### 字符字面量

在 C 语言中，字符字面量的类型为`int`, 而 C++ 中字符字面量的类型为`char`。

```cpp
/*
 * C 语言中输出4，C++ 输出1
 */
printf("%d", sizeof('1'));
```

### 赋值操作为左值

在 C 语言中，赋值运算符的返回值为右值，而 C++ 中其返回值为左值。

```cpp
/*
 * gcc 编译不通过
 * g++ 编译通过，输出“3 2 3”
 */
int a, b, c;
a = 1, b = 2, c = 3;
(a = b) = c;
printf("%d %d %d\n", a, b, c);
```
